# User Interviews

## Overview

Informal feedback sessions were conducted with students and developers familiar with AI productivity tools.

The purpose was to understand:
- AI tool usage patterns
- pricing confusion
- interest in spend optimization

---

## Participant Profiles

### Participant 1

- Computer Science student
- Uses ChatGPT and GitHub Copilot
- Interested in reducing recurring subscription costs

### Participant 2

- Beginner web developer
- Uses free and paid AI tools interchangeably
- Often unsure which plans are necessary

### Participant 3

- Small project collaborator
- Interested in team pricing comparisons
- Wanted simple savings recommendations

---

## Common Observations

### Overlapping Tools

Several users subscribed to multiple tools with similar functionality.

### Pricing Uncertainty

Users were often unclear about:
- plan differences
- enterprise benefits
- actual value gained

### Simplicity Was Important

Participants preferred:
- quick analysis
- minimal inputs
- easy-to-understand recommendations

---

## Feedback Received

### Positive Feedback

- clean UI
- straightforward recommendations
- useful public share links

### Suggested Improvements

- add more AI tools
- include real API pricing
- support PDF exports
- add historical tracking

---

## Key Takeaway

Users showed interest in lightweight tools that simplify AI software cost management without requiring complex setup.
